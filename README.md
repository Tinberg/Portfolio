## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:Tinberg/Portfolio.git
```
### ScrollReveal script or download
Use this in your your head element in HTML: 
```html
<script src="https://unpkg.com/scrollreveal"></script>
```
or Download it: 
```bash npm install scrollreveal --save```
